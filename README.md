# Global Pollution Analysis and Energy Recovery

This project analyzes global pollution data and predicts energy recovery using machine learning models.

## Objectives
- Perform data preprocessing and EDA
- Build Linear Regression model for energy prediction
- Build Logistic Regression model for pollution classification

## Tools Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-Learn
Jupyter Notebook

## Models
- Linear Regression
- Logistic Regression
